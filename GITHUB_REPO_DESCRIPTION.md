# GitHub Repository Description

## Description Utama (disarankan untuk kolom Description GitHub)

Sistem web prediksi spasial-temporal area risiko hotspot kebakaran hutan di Provinsi Riau berbasis ConvLSTM, NASA FIRMS, dan citra sekuensial H-6 sampai H0.

## Alternatif Description

1. Sistem web prediksi area risiko hotspot H+1 di Provinsi Riau menggunakan ConvLSTM, citra historis NASA FIRMS, heatmap, overlay peta, dan validasi geospasial.

2. Web app FastAPI untuk prediksi spasial-temporal hotspot kebakaran hutan Riau berbasis ConvLSTM historical risk patch dan data citra sekuensial.

3. Sistem prediksi risiko hotspot kebakaran hutan Riau dengan input 7 citra historis, model ConvLSTM patch 160x160, dan output heatmap serta overlay wilayah.

## About / Deskripsi Panjang untuk README atau GitHub About

Project ini merupakan sistem web prediksi area risiko hotspot kebakaran hutan di Provinsi Riau berbasis deep learning ConvLSTM. Sistem menerima tujuh citra historis berurutan dari H-6 sampai H0, melakukan preprocessing menjadi mask hotspot, menjalankan inferensi model historical risk patch 160x160, lalu menghasilkan prediksi H+1 dalam bentuk heatmap, overlay peta, binary mask, riwayat prediksi, dan paket unduhan output. Project juga menyertakan kode training, evaluasi, data pendukung geospasial, contoh input demo, serta artefak validasi untuk mendukung penyusunan skripsi/tugas akhir.
